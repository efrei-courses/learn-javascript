### Design-System

#### Buttons
```
Font
    - font-size: 12
Colors
    - background-color: #248F8
    - text-color: #616161
```

#### Messages
```
Font
    - font-size: 18
Colors
    - background-color: #0AA6FD OR #EFEFEF
    - text-color: #EBFAF8
```

#### Inputs
```
Font
    - font-size: 18
Colors
    - border-color: #BBBBBB
    - text-color: #BBBBBB
```
